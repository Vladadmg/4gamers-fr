        24/03/2004 <b>[Autre][PC]</b> - <b>CD Rom Virtuels</b> <font size="-2">par <a href="mailto:swapfilez@4gamers-fr.com?subject=CD Rom Virtuels" class=lien3>SwapFilez</a></font><p><font size="-2">Introduction:</font><br><table width="100%" border="1" cellpadding="8" cellspacing="0" bordercolor="#C8C8C8"><tr><td valign="top" bgcolor="#FFFFFF"><div style="font-family:Verdana; font-size=10pt;"><div align=justify>Les lecteurs Cd-rom virtuels<br />
<br />
Pour ceux qui ne connaissent pas encore le principe, un lecteur Cd virtuel est un logiciel qui fera croire à Windows que vous possédez presque autant de vrais lecteurs cd que vous voulez<br />
Le principe est très simple et plus rapide en exécution qu’un vrai Cd-rom.<br />
Tout ce que vous avez a faire c’est de <br />
1- Vous procurer le logiciel via le net (plusieurs versions sont payantes mais si vous cherchez bien vous trouverez des versions gratuites ou d’évaluation. )<br />
<br />
2- Faire l’installation et la plupart du temps, très peu de configuration est nécessaire…<br />
<br />
3- Par la suite grâce au logiciel vous pourrez vous ajouter des lecteurs cd virtuels à votre guise, copier le contenu entier de votre Cd-rom original vers le Cd-rom virtuel sans oublier de garder les mêmes noms de volumes … Certains logiciels s’occupent eux-mêmes de faire la copie correctement.<br />
<br />
4- Et c’est parti … lorsque vous lancerez votre jeux favori ça sera comme si le Cd original se trouvait dans votre lecteur cd physique … HE HE pas mal <br />
</div></div></td></tr></table><p><div align="justify">*****  Notez bien qu’il est préférable de créer votre lecteur virtuel et d’y copier votre jeu favori AVANT de procéder a l’installation CAR, il arrive que les développeurs de jeu incluent dans leurs scripts d’installation certaines variables qui, permettront au jeu, quand il sera lancé de savoir quel étais le répertoire source d’installation ce qui pourrait empêcher de faire fonctionner votre jeu …<br />
Un exemple vaut milles mots  ...<br />
<br />
Supposons que vous installez un jeu a partir de votre vrai cd-rom le ( d : ) par exemple<br />
Par la suite vous créer votre lecteur virtuel le ( E : ) <br />
Il y aura alors de fortes chances que votre jeu ne démarre par car des données sont souvent inscrites dans le registre et dans le répertoire d’installation qui indiquent que le jeu DOIT être lancé a partir du même lecteur CD que le lecteur à partir duquel s’est déroulé l’installation.<br />
C’est selon moi la seule chose a laquelle vous devrez porter attention.<br />
</div><br><p><font size="-2">Conclusion:</font><br><table width="100%" border="1" cellpadding="8" cellspacing="0" bordercolor="#C8C8C8"><tr><td valign="top" bgcolor="#FFFFFF"><div style="font-family:Verdana; font-size=10pt;"><div align=justify>Avantages d’utiliser les lecteurs Cd virtuels <br />
<br />
1- C’est qu’il y a de plus légal en matière de copyrights et de droits d’auteurs.<br />
<br />
2- C’est très pratique par exemple pour les familles … les enfants n’auront pas a manipuler les cd originaux <br />
<br />
3- Aussi pour ceux qui ne veulent pas risquer de briser leurs précieux originaux<br />
<br />
4- Et ça peut s’avérer aussi pratique pour certaines personnes qui ne possèdent pas les cd originaux … vous me suivez ...<br />
<br />
5- Peut être utilisé avec un serveur de cd rom et des clients qui se connectent via un réseau haut débit<br />
<br />
